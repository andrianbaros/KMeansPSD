# KMeansPSD
Repository ini berisi analisis data kasus kekerasan menggunakan algoritma K-Means Clustering. Data diolah berdasarkan kabupaten/kota dan tahun untuk mengelompokkan wilayah ke dalam tingkat kerawanan rendah, sedang, tinggi, dan sangat tinggi. Hasil analisis disajikan dalam bentuk visualisasi seperti bar chart, scatter plot.
